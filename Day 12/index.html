Display:Flex

Formally, the display property sets an element's inner and outer display types. The outer type sets an element's participation in flow layout; the inner type sets the layout of children. Some values of display are fully defined in their own individual specifications; for example the detail of what happens when display: flex is declared is defined in the CSS Flexible Box Model specification.

Display:Flex,Flex-Wrap

  • The flex-wrap CSS property sets whether flex items are forced onto one line or can wrap onto multiple lines. If wrapping is allowed, it sets the direction that lines are stacked.
  • The flex-wrap property can have one of the following values
  • nowrap
  • wrap
  • wrap-reverse

Display:Flex,Flex-Wrap:no-wrap

  • The flex items are laid out in a single line which may cause the flex container to overflow. The cross-start is the equivalent of inline-start or block-start, depending on the flex-direction value. This is the default value.

Display:Flex,Flex-Direction

  • The flex-direction property specifies the display-direction of flex items in the flex container.
  • The flex-direction property can have one of the following values
  • row
  • column
  • row-reverse
  • column-reverse

Display:Flex,Flex-Direction:row

  • The row value is the default value, and it displays the flex items horizontally

Display:Flex,Flex-Flow

  • The flex-flow property is a shorthand property for setting both the flex-direction and flex-wrap properties
  • The flex-flow property can have one of the following values
  • row-wrap
  • row-wrap-reverse
  • row-nowrap
  • column-wrap
  • column-wrap-reverse
  • column-nowrap

Display:Flex,Flex-Flow:row wrap;

  • The css property flex-flow: row wrap; is a shorthand for flex-direction and flex-wrap propertys on a flex container
