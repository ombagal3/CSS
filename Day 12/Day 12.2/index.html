Display:Flex(child property) Flex-Grow

  • The flex-grow CSS property sets the flex grow factor, which specifies how much of the flex container's positive free space, if any, should be assigned to the flex item's main size.
  • When the flex-container's main size is larger than the combined main sizes of its flex items, this positive free space can be distributed among the flex items, with each item's growth being their growth factor value as a proportion of the sum total of all the flex items' flex grow factors.
Flex-grow:Auto
Flex-grow:1;
Flex-grow: 2;

Display:Flex(child property) Flex-Shrink

  • The flex-shrink CSS property sets the flex shrink factor, which specifies how much of the flex container's negative free space, if any, should be assigned to the flex item's main size.
  • When the flex-container's main size is smaller than the combined main sizes of its flex items, this negative free space can be distributed among the flex items, with each item's shrinkage being their shrink factor value as a proportion of the sum total of all the flex items' flex shrink factors.
Flex-shrink:0;
Flex-shrink:1;
Flex-shrink: 2;

Display:Flex(child property) Align-Self: Flex-start;

  • The align-self CSS property allows the default alignment (or the one specified by align-items) to be overridden for individual flex items.
  • It accepts the same values as the align-items property, which means it can take the following values: auto, flex-start, flex-end, center, baseline, and stretch.
  • The cross-start margin edge of the flex item is flushed with the cross-start edge of the line.
